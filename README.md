# 求是 (QiuShi)

以哲学思辨为框架的 AI 思考伙伴。不是更快给答案，是帮你想清楚问题。

```bash
pip install qiushi
qiushi init        # 配置 API Key
qiushi             # 启动交互式 TUI
```

## 交互模式

直接运行 `qiushi` 进入交互式终端界面。启动后看到欢迎页 + 命令卡片，底部直接输入问题即可开始思辨。

```
你：该不该辞职
```

文字逐字出现，句号停顿。完成后渲染三段式彩色 Panel。

**内置命令（输入 / 查看）：**

| 命令 | 功能 |
|------|------|
| `/think` | 切换完整推理模式，展示概念炼金术/知识引用/矛盾检测等思考过程 |
| `/depth 1/2/3` | 设置分析深度（快速/标准/深度） |
| `/dialectic N <问题>` | 苏格拉底追问链（N=1-5 轮） |
| `/council` | 显示流派列表，按编号自选哲学人格辩论 |
| `/web <关键词>` | 联网搜索后思辨分析 |
| `/search <关键词>` | 在本地知识库中搜索 |
| `/profile` | 查看思维画像与决策执行率 |
| `/card` | 生成哲学思辨卡片 |
| `/note <内容>` | 记录笔记 |
| `/done` | 查看/标记决策已执行 |
| `/clear` | 清屏 |
| `/new` | 新对话 |

## 单次问答

```bash
# 基础问答
qiushi ask "压力很大"
qiushi ask "30岁转行来得及吗" --depth 3

# 查看推理过程
qiushi ask "不知道该选哪个offer" --think
qiushi ask "最近很迷茫" --explain

# 苏格拉底追问链
qiushi ask "该不该分手" --dialectic 3

# 多哲学人格辩论
qiushi ask "努力重要还是选择重要" --council 3

# 联网搜索
qiushi ask "2026年AI趋势" --web

# JSON 输出
qiushi ask "分析一下这个情况" --format json

# 输出到文件
qiushi ask "帮我理一下思路" --output result.md
```

## 核心机制

### 三段式思辨
每次回答走 **分析 → 反思 → 总结** 三步。先下判断、再自己反驳、最后落到行动。

- 普通模式：只显示【分析】+【总结】
- `/think` 模式：显示完整三段 + 思维诊断（知识引用、概念炼金术、矛盾检测等）

### 概念炼金术
你说"压力大"，它不会顺着你说是。它会追问：这个词准不准？是累、是焦虑、还是没方向？内置触发词：累、迷茫、压力、没意义、来不及等。

### 历史透视
你说"30岁转行"，它把你的个人焦虑放在时代背景里看——你的困境可能不是一个人的问题。支持触发：30岁、转行、35岁、裁员、买房等。

### 多哲学人格辩论 (`/council`)
从 25 个哲学流派中自选 2-5 个进行辩论（按时代/地区分组着色）。默认预设：斯多葛、辩证唯物、存在主义，但也支持自由组合——孔子、老子、尼采、萨特同台辩论。

### 苏格拉底追问链 (`/dialectic`)
第一轮回答 → 生成追问 → 等你回应 → 再分析 → 多轮后出辩证总结。不是一次对话，是帮你把问题想透。

### 语境比喻
60+ 条按场景过滤的比喻（事业、感情、管理），话说到了自然带出来，没到不硬塞。

## 场景

| 场景 | 适用 |
|------|------|
| `general` | 通用（默认） |
| `career` | 职业选择、转行、跳槽 |
| `relationship` | 感情、人际 |
| `management` | 管理、决策、团队 |

TUI 中使用 `/scenario <场景名>` 切换，单次使用 `--scenario` / `-c`。

## 用户画像与记忆

求是记录你的对话历史、矛盾演变和决策执行情况：

```bash
/profile            查看思维画像
/done               查看/标记决策已执行
/note <内容>        记录笔记
/history            查看对话历史
```

## 配置

```bash
qiushi init            # 交互式配置
~/.qiushi/config.json  # 配置文件
```

支持 LLM 提供者：**DeepSeek**（默认）、OpenAI、Anthropic Claude、Ollama（本地）、OpenRouter。

## 配色

低饱和度暖石色系：
- 【分析】灰蓝色 `#7B8B9B`
- 【反思】陶土色 `#B8846B`
- 【总结】石墨紫 `#6B5B7B`
- 品牌边框：石墨紫

## 项目目标

市面上不缺 AI 问答工具，缺的是能帮你**梳理思路**的东西。

求是不追求"多快好省地给出答案"，而是用哲学思辨框架让你自己想清楚：
- 你的困境主要矛盾是什么？
- 你漏掉了什么重要的角度？
- 最小的第一步是什么？
