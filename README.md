# 求是 (QiuShi)

以哲学思辨为框架的 AI 思考伙伴。不是回答问题，是帮你想清楚问题。

`qiushi ask "该不该辞职"` — 它不会直接告诉你答案。它会分析矛盾、追问盲区、最后让你自己得出判断。

## 快速开始

```bash
pip install qiushi
qiushi init        # 配置 API Key
qiushi chat        # 交互式对话
```

## 使用

```bash
# 单次问答
qiushi ask "压力很大"
qiushi ask "30岁转行来得及吗" --depth 3

# 苏格拉底追问链（多轮辩证）
qiushi ask "该不该分手" --dialectic 3

# 多智能体辩论（斯多葛 vs 辩证唯物 vs 存在主义）
qiushi ask "努力重要还是选择重要" --council 3

# 查看思考过程
qiushi ask "不知道选哪个 offer" --explain
qiushi ask "最近很迷茫" --think

# 思辨卡片
qiushi card "自由"
qiushi card "选择"

# 知识管理
qiushi knowledge add -p my-notes.md

# 交互模式
qiushi chat
```

## 核心机制

### 三段式思辨
每次回答都走 **分析 → 反思 → 总结** 三步。不是给一个结论，是先给判断再自己反驳。

### 概念炼金术
你说"压力大"，它不会顺着你说是。它会想："压力"这个词准不准？是累、是焦虑、还是没方向？

### 历史透视
你说"30岁转行"，它会把你的个人焦虑放在时代背景里看——你的困境可能不是一个人的问题。

### 语境比喻
60 条按场景过滤的比喻（事业、感情、管理），话说到了自然带出来，没到不硬塞。

### 苏格拉底追问链 (`--dialectic N`)
第一轮回答 → 生成追问 → 等你回应 → 再分析 → 多轮后出辩证总结。不是一次对话，是帮你把问题想透。

### 多智能体辩论 (`--council 2/3`)
让斯多葛、辩证唯物、存在主义三种哲学人格同时分析同一个问题，最后给出共识与分歧。

### 强制三段式
**分析** — 判断。**反思** — 找出逻辑漏洞和隐藏假设。**总结** — 综合后的最终判断。反思段默认不展示，用 `--think` 可见。

## 场景

`--scenario` / `-c` 参数按场景过滤比喻：

| 场景 | 适用 |
|------|------|
| `general` | 通用（默认） |
| `career` | 职业选择、转行、跳槽 |
| `relationship` | 感情、人际 |
| `management` | 管理、决策、团队 |

## 知识库

内置 43 篇哲学智慧，覆盖 8 个流派：

| 流派 | 篇数 | 来源 |
|------|------|------|
| 毛泽东 | 12 | 《实践论》《矛盾论》《论持久战》等 |
| 儒家 | 4 | 学习、领导力、仁爱、修身 |
| 道家 | 4 | 无为、对立统一、谦虚、返璞归真 |
| 孙子 | 3 | 谋略、灵活、时机 |
| 马克思 | 3 | 阶级史观、异化、经济基础 |
| 黑格尔 | 2 | 辩证法、扬弃 |
| 现代哲学 | 3 | 尼采、萨特、加缪 |
| 当代心理学 | 10 | 斯多葛、心流、反脆弱、思考快与慢等 |

可用 `qiushi knowledge add -p file.md` 添加自定义知识。

## 配置

```bash
qiushi init            # 交互式配置
~/.qiushi/config.json  # 配置文件
```

支持的 LLM 提供者：DeepSeek（默认）、OpenAI、Anthropic Claude、Ollama（本地）、OpenRouter / LiteLLM。

## 项目目标

市面上不缺 AI 问答工具，缺的是能帮你**梳理思路**的东西。

求是不追求"多快好省地给出答案"，而是用哲学思辨框架让你自己想清楚：
- 你的困境主要矛盾是什么？
- 你漏掉了什么重要的角度？
- 最小的第一步是什么？
